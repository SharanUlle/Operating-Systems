FCFS(First Come First Serve)
> It is a CPU process scheduling method in order to assign the process based on their arrival and burst time .
> It is a nonpreemptive method i.e. no other process are assigned to the CPU unless a process is fully completed 
> The first process assigned will be served first and is non preempted.
> The best case is calculated on the average Turnaround time and Waiting time.
